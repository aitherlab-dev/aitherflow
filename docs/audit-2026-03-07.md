## Критические проблемы

### [SEC-001] Command injection через test_hook_command
- **Файл:** `src-tauri/src/hooks.rs:101`
- **Серьёзность:** КРИТИЧЕСКАЯ
- **Описание:** Команда `test_hook_command` принимает произвольную строку `command` от фронтенда и передаёт напрямую в `sh -c` без валидации. Любой код с доступом к IPC (фронтенд, вредоносный плагин, скомпрометированный MCP-сервер) получает полный RCE.
- **Риск:** Произвольное выполнение команд от имени пользователя.
- **Статус:** By-design для десктопного Tauri-приложения (IPC доступен только собственному WebView).

### ~~[SEC-002] Command injection через platform::notify на macOS~~ DONE
- **Файл:** ~~`src-tauri/src/platform.rs:20-28`~~ удалён
- **Серьёзность:** КРИТИЧЕСКАЯ
- **Исправление:** Весь файл `platform.rs` удалён как мёртвый код (вместе с ARC-006).

### [BUG-001] Race condition в streamBuffer — потеря промежуточных chunk'ов — SKIP
- **Файл:** `src/stores/chatStore.ts`
- **Серьёзность:** КРИТИЧЕСКАЯ
- **Описание:** При получении `streamChunk` текст целиком перезаписывает `streamBuffer`. Если два chunk'а приходят быстрее одного кадра RAF (16мс), первый chunk теряется.
- **Статус:** Ложный позитив. CLI присылает аккумулированный текст (не дельту) — каждый chunk содержит весь текст с начала, перезапись корректна по дизайну.

### ~~[PRF-001] selectToolActivities — O(N) на каждый кадр стриминга~~ DONE
- **Файл:** `src/stores/chatStore.ts:1250-1259`
- **Серьёзность:** КРИТИЧЕСКАЯ
- **Исправление:** Добавлено кеширование по ссылочному равенству `messages` и `streamingMessage`. O(1) при неизменных данных.

## Что сделано хорошо

1. **validate_path_safe** — файловые операции корректно проверяют пути через canonicalize + whitelist
2. **Atomic writes** — все записи в конфиги через temp+rename, защита от повреждения
3. **spawn_blocking** — все Tauri-команды с FS/process обёрнуты корректно
4. **Параметризованные SQL** — все запросы к SQLite через `params![]`, SQL injection невозможна
5. **stderr buffer cap** — ограничение 64KB предотвращает DoS через память
8. **CSP** — `script-src` без `unsafe-inline`/`unsafe-eval`
9. **CI/CD** — clippy -D warnings + tsc --noEmit + eslint
10. **entry.file_type()** — вместо блокирующего `entry.path().is_dir()`
11. **useShallow** — корректно применяется в селекторах с массивами
12. **Lucide React** — все иконки из единой библиотеки
13. **.catch(console.error)** — нигде не найдено `.catch(() => {})`
