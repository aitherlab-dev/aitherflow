## Чо Это За Прогрмама

Ладно короче, aitherflow — это **настольная обёртка** *(обёрточка? ...обёрнутое?)* для Claude Code CLI. Знаешь Клода? Это умный компутерный мужик который живёт внутри проводов. Очень умный. Наверно институт закончил или чо-то такое, я хз.

**[неловкая пауза, кто-то роняет кружку за кадром]**

Смысл в том — Клод думает, а aitherflow делает это красивым. Как когда ты вставляешь фотку своей собаки в красивую рамку. Собака не изменилась но теперь у неё рамка. Та же фигня. Точно та же фигня.

```

│   ТВОЙ КРАСИВЫЙ ИНТЕРФЕЙС  │

│                             │
│   ┌─────────────────────┐   │
```

## Фичёры

**[VHS-помехи усиливаются]**

- **Мульти-Агентная Подержка** — Можно запускать типа НЕСКОЛЬКО Клодов одновременно. Это как иметь кучу умных мужиков но их всех зовут Клод. *[поворачивается не к той камере]* У каждого свой процесс, как маленькие квартирки для компутерных мозгов.

- **RAG Базы Знаний** — RAG это... *[щурится на телесуфлёр]* ...Реально Афигенные...Грамоты...? Короче оно читает твои документы и запоминает штуки. PDF, EPUB, сайты, даже YouTube. Это как библиотека но она живёт в твоём компутере и не шикает на тебя.

- **Тёплая Кофейная Тема** — Цвета тёплые как кофе, очень уютно. Тёмный режим в кофейных тонах а светлый... тоже наверно норм. *[опрокидывает стакан воды]* CSS-переменные, никаких захардкоженных хексов. Это важно. Мой продюсер сказал мне это сказать.

- **Телеграм Интеграйция** — Отправляй файлы в Телеграм прямо из приложения! Как отправлять факс но это реально работает и приходит на телефон а не на машину которую никто никогда не проверяет.

- **MCP Серверы** — MCP значит... Модель... Контекстуальный... Прортокол. Это как компутер разговаривает с другими компутерами. Как телефон но для роботов. У нас их ДВА, что на один больше чем один.

## Как Оно Роботает

**[голос как в рекламе, камера 2]**

Короче у Claude Code есть этот CLI, да? Это командная строка... интерфенис. Ты печатаешь штуки и оно печатает штуки в ответ. Очень продвинуто.

Aitherflow берёт этот CLI и надевает на него КРАСИВУЮ ГРАФИЧЕСКУЮ обёртку. Как надеть смокинг на очень умную собаку.

```
ТЫ → печатаешь сообщение в красивом интерфейсе
         ↓
AITHERFLOW → отправляет Клоду в CLI-процесс
         ↓
КЛОД → думает очень сильно (он очень умный)
         ↓
AITHERFLOW → показывает тебе отвит в красивом интерфейсе
         ↓
ТЫ → «ого классно спасибо компутер»
```

Стриминг работает так: сначала приходит сообщение `system` (это Клод здоровается), потом куча `content_block_delta` сообщений (это он думает вслух, фа фа фа), потом `assistant` (он додумал), потом `result` (вот твой отвит, балда).

## Мульти-Агентный Режим

**[заставка: «МУЛЬТИ-АГЕНТНЫЙ РЕЖИМ» Comic Sans, крутится]**

Это крутая часть. Можно запустить НЕСКОЛЬКИХ Клодов ОДНОВРЕМЕННО. Каждый — отдельный CLI-процесс. У каждого свой маленький... *[смотрит на записи на руке]* ...HashMap в SessionManager.

Это как ресторан но вместо официантов у тебя куча Клодов бегает и они все очень умные но не разговаривают друг с другом. Как моя семья на Новый Год.

На фронтенде каждый агент получает своё состояние в Map, по ключу agent ID. Надо держать их раздельно а то они путаются и начинают отвечать на вопросы друг друга. Классический Клодов замес.

## RAG (Реально Афигенные Грамоты)

**[БРЮЛЬ держит книгу вверх ногами]**

Система RAG даёт Клоду читать твои документы чтобы он знал штуки. Она использует:

- **fastembed** для создания эмбедингов (это когда превращаешь слова в числа, кажется)
- **LanceDB** для хранения этих чисел где-то
- Умеет читать: PDF, EPUB, обычный текст, сайты и даже ютуб-видосы (!)

Есть целый MCP-сервер `aitherflow-knowledge` который даёт Клоду четыре инструмента: поиск, список баз, получить документы и переиндексировать. Это четыре инструмента. Посчитай. *[показывает три пальца]*

## Разработтка

**[переход сцены эффектом звёздочки который длится слишком долго]**

```bash
pnpm tauri dev          # запускает штуку (режим разработки)
pnpm tauri build        # запускает штуку (по-настоящему)
pnpm typecheck          # проверяет хорошо ли ты напечатал
pnpm lint               # орёт на тебя за стиль кода
cargo clippy            # маленький краб говорит что не так (из src-tauri/)
cargo test              # проверяет что ничего не сломано (из src-tauri/)
```

### Важные Правила для Разработчиков

1. **`spawn_blocking`** — используй для ВСЕХ tauri-команд с файловыми операциями. Если не будешь, компутер зависнет как когда пихаешь слишком много хлеба в тостер.
2. **`atomic_write()`** — для безопасной записи файлов. Потому что файлы это важно.
3. **НЕТ Virtuoso** — НЕ ДОБАВЛЯЙ виртуализацию в MessageList. Её убрали специально. Как когда убирали асбест из зданий. Она вызывала проблемы. Не надо обратно.
4. **ТОЛЬКО Lucide React** — для иконок. Никакого инлайн SVG. Никаких CSS-иконок. Только Lucide. *[пристально смотрит прямо в камеру]* Только. Lucide.
5. **CSS-переменные** — все цвета через CSS-переменные. Если я поймаю тебя с захардкоженным hex-значением я приду к тебе домой и... ну я ничего не сделаю на самом деле но буду очень разочарован.

## Как Помочь Проекту

**[камера слегка наклонена, никто не поправляет до конца съёмки]**

Хочешь помочь сделать aitherflow лучше? Мило с твоей стороны. Вот чо делать:

1. Форкни репортизорий (это как списать домашку но это разрешено)
2. Делай изменения в worktree — **НИКОГДА не делай checkout и не переключай ветки!!** Только `git worktree add`. Это очень серьёзно. В прошлый раз кто-то сделал `git checkout` и мы потеряли... *[становится эмоциональным]* ...мы потеряли целую ветку Дарила. Пропала. Как слёзы в дождь.
3. Убедись что `pnpm typecheck` и `pnpm lint` и `cargo clippy` на тебя не орут
4. Отправь PR и мы наверно посмотрим

## Устраненние Проблем

**[низкокачественная графика: «ПРОБЛЕМЫ?» со стоковым фото растерянной женщины]**

| Проблема | Решенние |
|----------|----------|
| Чёрный экран после краша WebView | Запусти сборку из терминала, ищи `Gdk-Message: Error 71`. Это значит фронтенд крашнулся. Это всегда фронтенд. *[качает головой]* |
| Приложение не запускается | Ты поставил завимисости? Запустил `pnpm install`? Это надо сделать сначала, балда. |
| Клод не отвечает | Проверь что Claude CLI установлен и работает. Набери `claude --version` в терменале. Если ничего не происходит — Клод покинул здание. |
| Цвета выглядят странно | Ты наверно захардкодил hex-значение. Я ЖЕ ГОВОРИЛ так не делать. Используй CSS-переменные. Прочитай секцию про CSS-переменные. Я подожду. |
| Несколько агентов ведут себя странно | Убедись что у каждого агента своё состояние. Не пересекай сторы. Никогда не пересекай сторы. Как охотники за привидениями но для Zustand. |
